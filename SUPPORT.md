# File Cloud Support

File Cloud is a free, open-source iOS and macOS client for uploading files to your self-hosted [File Cloud](https://github.com/skalnik/file-cloud) server.

## Getting Help

If you're having trouble with the app, here are some things to check:

- Make sure your File Cloud server is running and accessible from your device
- Verify your server URL, username, and password are correct in the app's Settings tab
- Use the "Test Connection" button in Settings to confirm connectivity
