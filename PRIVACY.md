# File Cloud Privacy Policy

*Last updated: March 29, 2026*

## Overview

File Cloud is a file upload utility that lets you upload photos and documents
to your own self-hosted File Cloud server. Your privacy is important, and this
app is designed to collect as little data as possible.

## Data Collection

File Cloud does **not** collect, store, or transmit any personal data to the
developer or any third party. There are no analytics, tracking, or advertising
frameworks in the app.

## Data You Provide

- **Server credentials:** Your server URL and username are stored locally on
  your device using standard app storage. Your password is stored securely in
  the iOS/macOS Keychain.
- **Photos and files:** When you choose to upload a photo or document, the
  selected file is sent directly to the server you have configured. Files are
  temporarily copied to the app's local storage during upload and are not
  retained afterward.

## Where Your Data Goes

All file uploads and authentication requests are sent exclusively to the server
URL that you configure in the app's settings. No data is sent to the developer
or any other third party. You are responsible for the privacy and security
practices of the server you choose to use.

## Third-Party Services

File Cloud does not use any third-party services, SDKs, or analytics platforms.

## Security

Credentials are stored in the system Keychain. We recommend configuring your
server with HTTPS to ensure that uploads and credentials are encrypted in
transit.

## Children's Privacy

File Cloud does not knowingly collect any personal information from children.

## Changes to This Policy

If this policy is updated, the changes will be posted on this page with an
updated date.

## Contact

If you have questions about this privacy policy, you can reach the developer
via the contact information listed on the App Store.
